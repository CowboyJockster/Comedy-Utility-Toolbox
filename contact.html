How to Use This Tool

Purpose

The Contact page gives users a simple place to send questions, bug reports, suggestions, and new comedy tool ideas.

When To Use It

Use this page when a user finds something confusing, notices a broken link or tool problem, or has an idea that could improve the Comedy Utility Toolbox.

How To Use It

  1. Enter your name.
  2. Enter your email address.
  3. Write your message, suggestion, bug report, or tool idea.
  4. Click Send Message.

Example Input

Name: Test User
Email: fast.freddie268@gmail.com
Message: The Callback Mapper helped me connect two jokes, but I would like a print button.

Example Output

The page confirms the message was prepared. In a future live version, this button can be connected to email or a web form service.

Comedy Tip

Good feedback is specific. “Something broke” is a mystery. “The Save button did not work after I typed three callbacks” is a case file.

Message noted on this demo page. To send real messages from a live website, connect this form to an email or form service.
